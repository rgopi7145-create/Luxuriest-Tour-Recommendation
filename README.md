✨ Luxuriest Tour Recommendation System

A full-stack Luxury Tour Recommendation System designed to provide premium travel suggestions based on user preferences. This project combines a modern frontend with a robust backend to deliver a seamless booking and authentication experience.

🚀 Project Overview

The Luxuriest Tour Recommendation System helps users:

🔐 Register and Login securely

🧳 Explore luxury tour packages

📍 Get personalized tour recommendations

🏨 Book premium travel experiences

📅 Manage booking details

This project demonstrates full-stack development using modern technologies.

⚙️ How to Run the Project
1️⃣ Backend Setup
# Navigate to backend folder
cd backend

# Run Spring Boot Application
mvn spring-boot:run

Or run TourBotApplication.java directly from your IDE.

2️⃣ Frontend Setup
# Install dependencies
npm install

# Start development server
npm run dev
